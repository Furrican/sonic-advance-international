Updates

Upload Date: 04/04/2025

Version 4.1.3 release:
The "Quit the game and visit the game website?" message has been implemented in the game to avoid confusion with the regular "Quit the game?" message. Additionally, the GAME WEBSITE words has been translated into other languages.
任索嘉盒Ninesogabox has translated these words into Chinese.

Speaking of 任索嘉盒Ninesogabox, he also has noticed a forgotten translation about the stage select menu in the Chinese version of the game. This issue has been fixed.
